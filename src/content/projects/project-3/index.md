![Astro Micro](/astro-micro.jpg)

Astro Micro 是一个适用于 Astro 的无障碍主题。它基于 Mark Horn 的热门主题 Astro Nano 构建。和 Nano 一样，Micro 默认不安装任何前端框架。

Micro 新增了 [Pagefind](https://pagefind.app) 搜索、[Giscus](https://giscus.app) 评论等功能。完整变更列表请查看这篇 [博客文章](/blog/00-micro-changelog)。

Micro 依然保留了 Nano 的所有优点：完整的类型安全、站点地图、RSS 订阅，以及 Markdown + MDX 支持。样式基于 TailwindCSS，并预配置了系统、浅色和深色主题。

访问 [GitHub 上的 Astro Micro](https://github.com/trevortylerlee/astro-micro) 即可开始使用。

## 📋 功能

包含 [Astro Nano](https://github.com/markhorn-dev/astro-nano) 的全部功能，并额外提供：

- Pagefind 搜索
- Giscus 评论
- Callout 组件
- 目录组件
- 分页组件
- 404 页面

以及[更多内容](/blog/00-micro-changelog)。
